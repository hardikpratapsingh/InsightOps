# 🚀 InsightOps

InsightOps is a full-stack application built using React, Flask, PostgreSQL, Docker, and Jenkins.

## ✨ Features

* React Frontend
* Flask Backend API
* PostgreSQL Database
* Dockerized Application
* Jenkins CI/CD Pipeline
* Docker Compose Deployment

## 🛠️ Tech Stack

* React
* Flask
* Python
* PostgreSQL
* Docker
* Docker Compose
* Jenkins
* Git & GitHub

## ▶️ Run the Project

```bash
docker compose up -d
```

## 🔄 Jenkins Pipeline

The pipeline performs:

1. Checkout code from GitHub
2. Build backend Docker image
3. Build frontend Docker image
4. Deploy using Docker Compose
5. Verify successful deployment
